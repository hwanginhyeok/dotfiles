# 세무신고 필요 데이터 + 취합 방법

> SKILL STEP 2에서 사용. 프로파일에 맞는 항목만 추려 체크리스트로 제시한다.
> 핵심 원칙: **증빙 없는 비용 = 경비 불인정 = 세금 증가.** 적격증빙 회수가 절세의 8할.

---

## 1. 수입(매출) 데이터 — "얼마 벌었나"

| 데이터 | 취합처 | 비고 |
|--------|--------|------|
| 전자세금계산서 매출 | 홈택스 `My홈택스 > 전자세금계산서 조회` | 사업자간 거래 |
| 전자계산서 매출(면세) | 홈택스 동일 메뉴 | 면세 재화·용역 |
| 신용카드 매출 | 홈택스 `신용카드 매출자료 조회` / 여신금융협회 | POS·단말기 |
| 현금영수증 매출 | 홈택스 `현금영수증 조회` | 의무발행업종 필수 |
| 플랫폼 정산내역 | 배달앱(배민·쿠팡이츠), 오픈마켓(스마트스토어·쿠팡·11번가), PG사 | **국세청에 통보됨 — 누락 불가** |
| 현금·계좌이체 매출 | 사업용 계좌 거래내역 | 누락 시 적발 위험 |

> 홈택스 **모두채움/미리채워주는 서비스**가 매출 상당부분 자동 반영. 단, 플랫폼·현금매출은 본인 확인 필수.

---

## 2. 비용(매입·경비) 데이터 — "얼마 썼나" (절세 핵심)

### 적격증빙 4종 (이게 있어야 경비 인정)
1. **세금계산서 / 계산서** (사업자간 거래)
2. **신용카드 매출전표** (사업용 카드)
3. **현금영수증** (지출증빙용 — 사업자번호로 발급)
4. **원천징수영수증** (인건비)

| 비용 항목 | 취합처 | 주의 |
|-----------|--------|------|
| 매입 세금계산서 | 홈택스 `전자세금계산서 조회` | 매입세액공제 핵심 |
| 사업용 신용카드 사용내역 | 홈택스 `사업용 신용카드 등록·조회` | **카드 등록하면 자동 집계** |
| 현금영수증(지출증빙) | 홈택스 `현금영수증 조회` | 개인 소득공제용 아님, 사업자용으로 |
| 인건비(직원) | 원천징수 신고분, 4대보험 | 급여대장·근로계약 |
| 인건비(프리랜서 외주) | 3.3% 원천징수영수증 | 지급명세서 제출 |
| 임차료 | 임대차계약서 + 세금계산서/금융증빙 | |
| 4대보험 사업자부담분 | 공단 납부내역 | 경비 인정 |
| 차량 유지비 | 카드/현금영수증, 보험·정비 | 업무사용 비율 |
| 감가상각 자산 | 기계·차량·비품 취득 세금계산서 | 내용연수 상각 |
| 이자비용 | 사업용 대출 이자납입증명 | 사업관련분만 |
| 통신·공과금·소모품 | 사업장 명의 청구서·카드 | |

### 비용 누락 방지 체크
- ☐ 사업용 신용카드를 홈택스에 **등록**했는가? (안 하면 일일이 입력)
- ☐ 사업용 계좌를 **등록**했는가? (복식부기 의무자는 미등록 시 가산세)
- ☐ 3만원 초과 지출에 적격증빙 받았는가? (적격증빙 없으면 증빙불비가산세 2%)
- ☐ 인건비 지급명세서·원천세 신고 했는가? (안 하면 경비 부인 + 가산세)

## 3. 종합소득세 추가 데이터 — "공제받을 것"

| 데이터 | 취합처 |
|--------|--------|
| 부양가족(인적공제) | 가족관계, 소득요건(연 100만원/근로500만원 이하) |
| 국민연금 납부액 | 공단/홈택스 (전액 소득공제) |
| 노란우산공제 납입액 | 중소기업중앙회 납입증명 (소득공제) |
| 연금저축·IRP 납입액 | 금융기관 (세액공제) |
| 기부금 | 기부금영수증 |
| 기타 소득 | 근로·이자·배당·연금·기타소득 (합산/분리 판정) |
| 기납부세액 | 원천징수된 3.3%, 중간예납액 (환급 가능) |

---

## 4. 부가세 신고 데이터

- **매출세액**: 위 1번 수입 데이터 (과세/면세 구분)
- **매입세액**: 매입 세금계산서 + 사업용 카드/현금영수증 매입분
- 의제매입(음식점 등 면세농산물), 신용카드매출 세액공제, 고정자산 매입
- 홈택스 부가세 신고 시 전자세금계산서·카드·현금영수증 **자동 불러오기** 지원

---

## 5. 취합 우선순위 (실무 순서)

1. **홈택스 로그인 → 사업용 카드·계좌 등록 확인** (안 했으면 지금 등록)
2. **My홈택스에서 매출/매입 자동집계 다운로드** (전자세금계산서·카드·현금영수증)
3. **플랫폼 정산내역 다운로드** (배달앱·오픈마켓·PG)
4. **카드사·은행에서 사업용 카드·계좌 내역** (자동집계 누락분 보완)
5. **인건비·임차료·4대보험 등 정기지출 정리**
6. **공제 서류**(연금·노란우산·기부 등) 모으기
7. 엑셀/워크시트(`templates/worksheet.md`)에 항목별 합계 정리

> 자동집계 ≠ 완벽. 현금매출·일부 플랫폼·증빙 없는 비용은 본인이 채워야 한다.
